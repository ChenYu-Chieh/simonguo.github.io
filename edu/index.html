  • 课程时间：2017年5月8日 14:00 - 17:00

  • 第一课：前端体系及基础知识

    • 目标：对 WEB 前端开发有一定认识
    • 计划：
      • WEB 发展历史
      • 前端范围及知识体系，学习曲线

      • HTML / CSS 基础介绍

  • 第二课：前端开发常用工具库

    • 目标：完成一个响应式的网站

  • 第四课：搭建一个 SPA 网站

    • 目标：完成一个 SAP 架构的网站

      • SPA 简介
